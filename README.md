# SDPPS

# Content Classification Module 

This project contains models, scripts, and cleaned datasets for content classification tasks (educational, Q&A, YouTube, BBC, IMDb, etc.).

## Project Structure
- `Cleaned_Datasets/` → cleaned datasets ready to use
- `Models/` → trained models (SVM, TF-IDF, etc.)
- `SSDPP-extension/` → browser extension code
- `*.py` → training and prediction scripts

## Datasets
Some original/raw datasets are **not included** in this repository due to size restrictions.  

To run the project fully, download the datasets from https://drive.google.com/drive/folders/151-9hW3eBAmQcNFhz7TyrCFU1ppGVxIW?usp=drive_link

and place them in:content_classification/Datasets/


Ensure that the folder structure remains the same.

## Usage
1. Place datasets in the correct folder as above.
2. Activate your virtual environment:  
3. Install requirements (if any)  
4. Run training or prediction scripts as needed.
